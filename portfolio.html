Project 01

종루이코리아 MES
생산관리(PL) 시스템

설비 · 기준정보 · 생산계획부터 실적까지 — 제조 현장 전체를 잇는 생산관리 시스템.

DWG NO종루이 코리아 MES
DOMAIN제조 · MES
ROLEFull-Stack · 설계 ~ 개발
PERIOD2024.11 – 2025.07
STACKJava · Spring Boot · Vue.js · MySQL

설비를 관리 단위로 두고, 그 아래에 공정을 매핑할 수 있도록 구조를 설계했습니다. 원소재·반제품·완제품으로 이어지는 BOM 체계를 만들어 “무엇을 어떤 설비·공정에서 가공해 무엇이 되는지”를 데이터로 표현했고, 이를 기반으로 생산요청부터 생산계획·지시, 일일 단위의 실행, 현장에서의 실적 입력(LOT_NO 채번), 작업일보까지 — 생산의 전체 흐름이 하나의 시스템 안에서 연결되도록 만들었습니다. 다국어 화면은 서버 기동 시 데이터를 클라이언트에 캐싱해, 화면 전환마다 별도 호출 없이 메뉴와 라벨이 자동으로 매핑되도록 설계했습니다.

01 · Equipment

생산 설비 관리

생산 설비를 등록하고 하위에 공정을 매핑. 설비별 수리 내역과 가동중단 이력을 등록·조회할 수 있도록 설계했습니다.

02 · Master Data

기준정보 관리

거래처(매출처/매입처), BOM, 완제품·반제품·원소재를 설계. 각 제품이 어떤 자재와 공정/설비로 만들어지는지 정의했습니다.

03 · i18n

다국어 관리

다국어 관리 화면을 개발하고, 서버 기동 시 데이터를 localStorage에 세팅해 화면 이동 시 메뉴·라벨이 자동으로 매핑되도록 구현했습니다.

04 · Production

생산관리

생산요청 → 생산계획 → 생산지시 → 일일생산계획 → 생산실적(LOT_NO) → 현황조회 → 작업일보까지, 생산 전 과정을 화면과 데이터로 연결했습니다.

  • 설비별 LOT_NO 자동 채번으로 생산 단위 추적성 확보
  • 다국어 데이터 캐싱으로 화면 전환 시 API 호출 최소화
Project 02

My KT 앱
SHOP 메뉴 개발

상품 조회부터 결제까지 — 본인인증 · 주소조회 · 결제 · 사내 코어 시스템을 잇는 커머스 프로세스.

DWG NOKT-SHOP
DOMAIN통신 · 커머스
ROLEFull-Stack · SHOP/운영자 개발 + 외부연동
PERIOD2023.10 – 2024.09
STACK※ 사용 기술 입력

My KT 앱의 SHOP 메뉴에서 무선·유선·액세서리 상품의 조회부터 결제까지 이어지는 구매 프로세스를 개발했습니다. 상품·주문·업체를 관리하는 운영자 프로젝트를 별도로 구축해, 운영자가 등록한 상품이 고객용 SHOP 홈페이지에 노출되도록 연동했습니다. 주문 과정에는 PASS·네이버·SMS·신용카드 본인인증으로 받은 CI 정보를 통해 고객을 조회·인증하는 단계, 행정안전부 API를 통한 주소 검색, token·카드정보·CI를 SmartroPAY에 전달해 결제를 처리하는 단계가 차례로 연결됩니다. 주문이 확정되면 SHOP과 KOS Core 시스템 간 연동 프로그램을 통해 사내 코어 시스템에도 정보가 반영됩니다.

01 · Purchase Flow

상품 조회 ~ 결제 프로세스

무선·유선·액세서리 상품의 목록·상세 조회부터 결제까지, 하나의 구매 흐름으로 이어지는 화면과 로직을 개발했습니다.

02 · Admin Sync

운영자 프로젝트 연동

상품·주문·업체를 관리하는 별도의 운영자 프로젝트를 구축하고, 등록된 상품이 고객용 SHOP에 실시간으로 노출되도록 연동했습니다.

03 · Identity

본인인증 연계

PASS · 네이버 · SMS · 신용카드 4종의 본인인증을 연동하고, CI 정보로 고객 정보를 조회해 인증 여부를 확인했습니다.

04 · Address API

주소 검색 연동

행정안전부 도로명주소 검색 API를 연동해 배송/설치 주소를 조회하고 주문 정보에 저장했습니다.

05 · Payment

SmartroPAY 결제 연동

결제 시 token, 카드정보, CI정보를 SmartroPAY에 전달하고 결과를 받아 주문 상태에 반영하는 프로세스를 개발했습니다.

06 · Core Sync

SHOP – KOS Core 연동

SHOP과 KOS Core 간 연동 프로그램을 개발해, 주문/계약 정보가 사내 코어 시스템과 동기화되도록 구성했습니다.
